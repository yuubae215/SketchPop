# モダン3D CAD UX改善提案

## 概要
現在のSketchPopをプロフェッショナルな3D CADソフトウェア（Fusion 360、SolidWorks、Onshapeなど）のUXに近づけるための包括的な改善提案です。

---

## 🎯 優先度A: 即座に効果が高い改善

### 1. ViewCube（ビューキューブ）の実装
**目的**: 3Dビューポートの方向を直感的に操作・表示

**実装内容**:
- 画面右上にインタラクティブな3Dキューブを配置
- 各面をクリックで標準ビュー（正面/背面/左/右/上/下）に切り替え
- 角をクリックでアイソメトリックビュー
- 現在の視点方向を視覚的に表示
- スムーズなアニメーション遷移

**技術的実装**:
```javascript
class ViewCube {
  // Three.jsで小さなシーンを作成
  // メインキャンバスの右上にオーバーレイ
  // カメラ方向と同期
}
```

**参考**: Autodesk製品の標準機能

---

### 2. ステータスバーの追加
**目的**: 現在の状態、座標、統計情報を常時表示

**実装内容**:
- 画面下部に固定バー
- 表示項目:
  - カーソル座標 (X, Y, Z)
  - 選択中のオブジェクト情報
  - ポリゴン数/頂点数
  - 現在のモード（スケッチ/押し出し/選択）
  - カメラタイプ（透視/平行）
  - 操作ヒント

**UIデザイン**:
```css
#status-bar {
  height: 28px;
  background: #1e1e1e;
  border-top: 1px solid #404040;
  display: flex;
  align-items: center;
  padding: 0 15px;
  gap: 20px;
  font-size: 12px;
}
```

---

### 3. プロパティパネルの強化
**目的**: 選択中のオブジェクトの詳細情報と編集機能

**実装内容**:
- 右サイドバーを拡張（現在60px → 280px）
- 折りたたみ可能なセクション:
  - **Transform**: 位置(X,Y,Z)、回転、スケール
  - **Dimensions**: 幅、高さ、奥行き（数値入力可能）
  - **Appearance**: 色、透明度、マテリアル
  - **Information**: 名前、タイプ、作成日時
- 数値入力フィールドでリアルタイム更新
- スライダーとテキスト入力の両対応

**構造**:
```html
<div id="properties-panel">
  <div class="property-section">
    <h4>Transform</h4>
    <div class="property-row">
      <label>X</label>
      <input type="number" step="0.1" />
    </div>
    <!-- ... -->
  </div>
</div>
```

---

### 4. グリッドとスナップ機能の改善
**目的**: 精密な配置をサポート

**実装内容**:
- グリッド表示のトグル機能
- グリッドサイズの調整（1, 5, 10, 25, 50mm）
- スナップ機能:
  - グリッドスナップ
  - オブジェクトスナップ（エッジ、頂点、面の中心）
  - 角度スナップ（15°, 30°, 45°, 90°）
- ステータスバーにスナップ設定を表示

**UI要素**:
```javascript
// ツールバーボタン
[Grid On/Off] [Snap ▼]
  └→ Grid Snap
  └→ Object Snap
  └→ Angle Snap
```

---

### 5. アクシストライアド（軸表示）
**目的**: 現在の作業平面と軸方向を常に表示

**実装内容**:
- 画面左下に固定表示
- X軸（赤）、Y軸（緑）、Z軸（青）
- カメラ回転に追従
- クリックで各軸方向にカメラ整列

## 🎨 優先度B: UXの洗練化

### 6. トップメニューバーの実装
**目的**: プロフェッショナルなアプリケーション構造

**実装内容**:
```
File | Edit | View | Create | Modify | Tools | Help
```

**各メニュー内容**:
- **File**: New, Open, Save, Export (STL, OBJ)
- **Edit**: Undo, Redo, Copy, Paste, Delete
- **View**: ViewCube, Grid, Axis, Wireframe, Shaded
- **Create**: Sketch, Box, Sphere, Cylinder
- **Modify**: Extrude, Move, Rotate, Scale, Mirror
- **Tools**: Options, Preferences
- **Help**: Documentation, Keyboard Shortcuts

---

### 7. コマンドパレット（クイック検索）
**目的**: キーボード中心の効率的な操作

**実装内容**:
- ショートカット: `Ctrl/Cmd + K` または `Ctrl/Cmd + P`
- ファジー検索で全コマンドにアクセス
- 最近使用した履歴
- キーボードショートカット表示

**参考**: VS Code、Figma、Fusion 360のクイックアクセス

**UI**:
```
┌─────────────────────────────────┐
│ > sketch                        │
├─────────────────────────────────┤
│ ✏️  Create Sketch           S   │
│ 📦  Extrude                 E   │
│ ⬜  Create Rectangle        R   │
│ 👁️  Toggle Wireframe       W   │
└─────────────────────────────────┘
```

---

### 8. コンテキストメニュー（右クリックメニュー）
**目的**: 状況に応じた適切な操作を提供

**実装内容**:
- オブジェクト選択時:
  - Edit
  - Duplicate
  - Delete
  - Hide/Show
  - Rename
  - Properties
- 空間クリック時:
  - Paste
  - Create New
  - View Options
  - Grid Settings

---

### 9. ヒストリー/タイムライン
**目的**: 操作履歴の可視化と編集

**実装内容**:
- 画面下部または左サイドバー下部
- 全操作のリスト表示:
  - Sketch-1
  - Extrude-1 (10mm)
  - Move-1
  - Extrude-2 (5mm)
- 各項目をクリックで編集
- ドラッグ&ドロップで順序変更
- 右クリックで削除/無効化

**参考**: Fusion 360のタイムライン機能

---

### 10. ホーム/フィット機能
**目的**: ワンクリックで全体表示

**実装内容**:
- ViewCubeの下に「🏠 Home」ボタン
- 全オブジェクトがフィットするようカメラを調整
- ショートカット: `F` キー

## 🔄 優先度F: ワークフロー改善

### 23. Undo/Redo スタック
**実装内容**:
- コマンドパターンで実装
- 全操作を記録
- メモリ効率的なスナップショット

**構造**:
```javascript
class CommandManager {
  constructor() {
    this.history = [];
    this.currentIndex = -1;
  }

  execute(command) {
    command.execute();
    this.history.push(command);
    this.currentIndex++;
  }

  undo() { /* ... */ }
  redo() { /* ... */ }
}
```

## 📝 まとめ

この提案書では、SketchPopを本格的な3D CADツールに進化させるための26の改善策を提示しました。

**最重要項目**:
1. ViewCube - 3Dナビゲーションの標準
2. ステータスバー - 常時情報表示
3. プロパティパネル - 精密な数値制御
4. Undo/Redo - 基本的なワークフロー

段階的に実装することで、プロフェッショナルなCADソフトウェアに匹敵するUXを実現できます。
